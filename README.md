## ✨ 最近在做的事

### 🚁 基于视觉的无人机自主追踪与降落
用 Android 手机作为边缘端，结合 DJI Mobile SDK、OpenCV 和 YOLOv8，做一个在 GPS 拒止场景下也能工作的视觉控制原型。  
目前主要在推进：目标识别、实时预览、追踪控制、降落流程联调。

### 🏥 多模态医疗风险预测与报告生成系统
结合临床文本、时序指标和大模型能力，做医疗风险预测与可视化报告生成。  
这一类项目让我越来越喜欢“AI 真正落地到现实问题里”的感觉。

### 💬 LLM / 医疗对话评测与应用
对大模型在医疗场景中的回答质量、稳定性和实用性很感兴趣，也做过相关的评测、分析和系统实现。
